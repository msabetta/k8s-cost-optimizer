# Roadmap

## 🎯 Vision

Diventare una piattaforma FinOps intelligente per Kubernetes, con ottimizzazione automatica dei costi.

---

## 🟢 Fase 1 – MVP (completata / in corso)

- [x] Collector Prometheus
- [x] Cost calculation engine
- [x] REST API
- [x] Basic frontend dashboard
- [x] Docker + Kubernetes deploy

---

## 🟡 Fase 2 – Smart Optimization

- [ ] Rightsizing automatico
- [ ] Idle resource detection avanzato
- [ ] Recommendation engine migliorato
- [ ] API filtering (namespace, workload)

---

## 🔵 Fase 3 – AI & Forecasting

- [ ] Modelli ML per cost prediction
- [ ] Anomaly detection avanzata
- [ ] Alerting (Slack, Email)

---

## 🟣 Fase 4 – Enterprise Features

- [ ] Multi-cluster support
- [ ] Multi-cloud pricing (AWS, GCP, Azure)
- [ ] RBAC & authentication
- [ ] Billing dashboard

---

## 🔴 Fase 5 – SaaS Platform

- [ ] Multi-tenant architecture
- [ ] Web dashboard avanzata
- [ ] Self-service onboarding
- [ ] Subscription model

---

## 🚀 Go-To-Market

1. Target: Startup Kubernetes-native
2. Open source core + premium features
3. Integrazione con:
   - Prometheus
   - Grafana
   - Cloud providers

---

## 📈 Metriche di successo

- Riduzione costi cluster (%)
- Numero cluster monitorati
- Monthly active users
- Conversione free → paid
